#AHK Productivity Suite
A custom AutoHotkey system for improving workflow, navigation, and system control on Windows.

1. Overview

This script provides a fast, lightweight set of hotkeys and automation tools to enhance everyday Windows use.
It includes custom app launchers, system toggles, browser shortcuts, and a collection of typing helpers.

The script uses privacy-safe paths and is structured into clear blocks for easy editing and expansion.

2. Features

• CapsLock remapped to Alt (with a shortcut to toggle CapsLock normally)
• Global shortcuts for refreshing, closing apps, and reloading the script
• Quick-launch hotkeys for frequently used programs
• System toggles: volume *more can be added
• Autocorrect with comprehensive list for English
• Autocorrect with small list for Norwegian

3. Usage

Install AutoHotkey v1 *ahk v2 is in making

Download this repository

Right-click AHK Main.ahk → Run as administrator

The script now runs in the system tray and all hotkeys are active

To reload the script, use the built-in hotkey (Win + your reload key).

4. File Structure
AHK Main.ahk      ; primary automation script
README.txt        ; this file

5. Customization

The script is fully modular.
Each functional category is separated by section dividers, making it easy to:

Change programs launched

Add new hotkeys

Remove unwanted modules

Port parts of the script into AHK v2 if desired
